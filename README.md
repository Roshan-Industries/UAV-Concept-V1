# UAV Concept V1

## Overview
UAV Concept V1 is a fixed-wing unmanned areal vehicle concept designed in Fusion 360. The design includes a streamlined fuselage, taped wings, V-tail, pusher propellor configuration, nose sensor turret, and underwing payload mounting points.
This project focuses on:
-Aircraft CD design 
-UAV configuration development
-Aerodynamic geometry 
-Engineering documentation


## Objectives
-Design a complete  UNV concept model
-Practice aircraft CAD modelling in Fusion 360
-Apply basic aerospace design thinking
-Prepare a portfolio-ready engineering project
